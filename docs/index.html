About the App

Quran Revision Tracker is an offline-first mobile application designed to help Muslims track their Quran memorization progress. The app features:

  • Track memorization progress for all 114 surahs
  • Smart revision scheduling
  • Offline functionality - no internet required
  • Export/import data for backup
  • Privacy-focused - all data stored locally
